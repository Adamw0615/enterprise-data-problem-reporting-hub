# Enterprise Data Problem Reporting Hub

## Overview

A SharePoint, Power BI, and Power Automate solution designed to simulate enterprise data problem management reporting.

## Business Problem

Organizations often struggle to track data issues, identify root causes, monitor SLA compliance, and provide executive visibility into remediation efforts.

## Solution

This project provides:

- Centralized issue tracking using SharePoint
- SLA monitoring and aging analysis
- Root cause reporting
- Executive reporting dashboards
- Automation opportunity analysis
- Automated notifications using Power Automate

## Tools Used

- SharePoint Online
- Power BI
- Power Automate
- Excel
- GitHub
